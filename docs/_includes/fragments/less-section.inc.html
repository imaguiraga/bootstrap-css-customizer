All checked Components will be compiled into a readable bootstrap.css and a minified bootstrap.min.css in addition there will be variables.less . We recommend you use the minified version in production.

If only one set of components (i.e button) is checked less be compiled into a readable bootstrap_[component].css and a minified bootstrap_[component].min.css in addition there will be variables_[component].less . We recommend you use the minified version in production.
